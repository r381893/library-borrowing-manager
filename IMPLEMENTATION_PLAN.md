# 圖書館借書清單管理系統 - 開發計畫

## 1. 專案目標
建立一個現代化、高質感的「圖書館借書管理系統」前端網頁。
讓使用者可以輕鬆匯入現有的 Excel 清單，補全作者資訊，並提供強大的分類、排序與統計功能。

## 2. 技術架構
*   **前端框架**: React (使用 Vite 建置) - 快速、響應式且易於擴充。
*   **語言**: JavaScript (ES6+)
*   **樣式**: Vanilla CSS (原生 CSS) - 採用 CSS Variables 實現主題切換 (深色/淺色模式)，打造 Premium 質感 (Glassmorphism 玻璃擬態風格)。
*   **資料儲存**: 
    *   初期：使用瀏覽器 `localStorage` 儲存資料，並提供 JSON 檔案匯入/匯出功能。
    *   資料來源：將您現有的 `圖書館借書清單.xlsx` 轉換為初始資料庫。

## 3. 核心功能規劃

### A. 書籍管理 (CRUD)
*   **列表檢視**: 以卡片 (Card) 或清單 (List) 模式顯示書籍。
*   **新增/編輯**: 補完書籍資訊，最重要的欄位包含：
    *   書名 (Title)
    *   作者 (Author)
    *   借閱狀態 (在庫/已借出/已歸還)
    *   借閱日期
*   **刪除**: 移除書籍。

### B. 智慧排序與搜尋
*   **筆畫排序**: 實作「作者姓名筆畫排序」功能 (使用 `localeCompare('zh-TW-u-co-stroke')`)，讓您方便依照傳統字典序查找作者。
*   **即時搜尋**: 輸入關鍵字即時篩選書名或作者。

### C. 統計儀表板 (Dashboard)
*   **作者統計**: 顯示「目前有哪些作者」，以及「每位作者擁有幾本書」。
*   **借閱狀態**: 統計多少書已借出、多少在庫。

## 4. 介面設計風格 (Aesthetics)
*   **配色**: 採用深色系靛藍/紫色漸層背景，搭配半透明玻璃質感卡片 (Glassmorphism)。
*   **字體**: 使用現代無襯線字體 (`Inter` 或 `Noto Sans TC`)，確保閱讀舒適。
*   **互動**: 滑鼠懸停特效 (Hover effects)、平滑過場動畫。

## 5. 執行步驟
1.  **資料轉換**: 將您目前的 Excel 轉為 JSON 格式，並手動/自動補上部分作者範例 (因目前 Excel 只有書名，需建立欄位讓您輸入作者)。
2.  **專案建置**: 初始化 Vite + React 專案。
3.  **核心開發**: 實作主畫面、書籍卡片、編輯表單。
4.  **功能實作**: 加上筆畫排序邏輯、統計邏輯。
5.  **優化**: 調整 UI 細節與動畫。
